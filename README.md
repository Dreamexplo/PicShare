# PicShare - 轻量级电子相册

拍摄者上传照片，自动生成精美版式相册，通过链接或二维码分享给他人。

## 功能特性

- **三种默认版式**：瀑布流（Masonry）、照片墙（Grid）、画册（Book）
- **一键上传**：拖拽或点击上传，自动生成缩略图
- **二维码分享**：自动生成分享链接和二维码
- **自由下载**：单张下载或全部打包（ZIP）
- **密码保护**：可选访问密码
- **无需注册**：访客无需登录即可浏览和下载

## 技术栈

| 模块 | 方案 |
|------|------|
| 前端框架 | Next.js 14 (App Router) |
| 样式 | Tailwind CSS |
| 数据库 | SQLite (better-sqlite3) |
| 图片处理 | sharp (缩略图生成) |
| 二维码 | qrcode (客户端生成) |
| 打包下载 | JSZip (客户端打包) |

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 初始化数据库

```bash
npm run db:init
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 即可使用。

### 4. 构建生产版本

```bash
npm run build
npm start
```

## 使用流程

1. 打开首页，点击「开始使用」或「创建我的相册」
2. 填写相册标题，选择版式（瀑布流/照片墙/画册），可选设置访问密码
3. 创建后进入相册管理页，拖拽或点击上传照片
4. 上传完成后，复制分享链接或扫码分享给他人
5. 访客打开链接即可按所选版式浏览，可单张下载或全部打包

## 版式说明

- **瀑布流**（Masonry）：照片按原始宽高比错落排列，适合混合尺寸
- **照片墙**（Grid）：统一网格裁切，整齐美观
- **画册**（Book）：水平翻页式浏览，模拟画册效果
